# 编辑质量标准

这五项按顺序决定文章能否交付：

1. **准确**：事实可核对；来源真的支持正文主张；事实、推断、意见分得开。
2. **观点**：作者对题目有明确判断，也说明适用边界，不把共识换句话重复。
3. **有用**：读者得到可执行步骤、判断方法、具体例子或新的解释框架。
4. **合声**：语气符合账号与作者，个人经历只来自用户材料，绝不虚构。
5. **好读**：结构自然、段落各有任务、语言直接，删掉套话、重复和无效标题。

工具能发现禁用词、句式过齐和可读性风险，但不能判断事实、洞察与实用性。工具分数只
用于定位问题，不能替代编辑判断，也不能为了分数制造碎句、情绪配额或随机离题。

交付前自问：读者能否指出这篇文章最重要的一个判断？关键事实能否追到来源？删掉任一
节会不会影响论证？文中是否出现了作者没有提供的亲历？如果答案不理想，定点修改。
